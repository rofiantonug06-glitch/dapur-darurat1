Aplikasi web untuk mencari resep berdasarkan bahan-bahan yang tersedia di dapurmu.


package-lock.json dan package.json
`npm init -y`

Tambah React & Vite
`npm install react react-dom`
`npm install -D vite @vitejs/plugin-react`
`npm install -D @types/react @types/react-dom`


Tambah Tailwind
`npm install -D tailwindcss postcss autoprefixer`
`npx tailwindcss init -p`


`git clone `
`cd dapur-darurat1`


Start with  
`NPM Install`
  
Run with (for development purposed)  
`NPM Run Dev`
